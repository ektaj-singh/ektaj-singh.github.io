What does it mean to display inline vs inline blocks?

In HTML and CSS the display property controls how an element is made on a web page. Two common values for the display property are "inline" and "inline-block".

When an element has a display property set to inline, it means that the element will be displayed inline with the surrounding content, and will not create a new line, so one after the other Examples of elements that are displayed inline are links, images, texts and spans.

On the other hand, when an element has a display property set to inline block, it will be displayed inline with the surrounding content, but will also have a height and width and can also have margins and paddings applied to it. Which allows it to behave like a block level element, but still be displayed inline. Examples of elements that are displayed as inline block are input fields and buttons.

In summary, inline elements are displayed inline with the surrounding content and do not create a new line, while inline block elements are displayed inline with the surrounding content but can also have height, width, margin and padding properties applied to them.
